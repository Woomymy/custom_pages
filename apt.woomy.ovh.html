Woomy's debian repository

Installing the repo

1: Adding the GPG key

# curl https://apt.woomy.ovh/key.gpg | apt-key add -

2: Adding the repository in /etc/apt/sources.list.d/

# apt install apt-transport-https
# echo "deb https://apt.woomy.ovh/ bullseye main" >> /etc/apt/sources.list.d/woomy-packages.list

3: Optional: Browse repo content:

Open https://apt.woomy.ovh/browser to watch the site content
